# **Nepal Income Tax Calculator**

A clean, minimal, and responsive one-page web application to help users in Nepal calculate their annual income tax based on the latest government regulations.  
This tool provides a clear breakdown of tax liability, shows which tax slabs apply, and offers suggestions for optimizing tax savings through eligible deductions.

## **Features**

* **Dynamic Tax Calculation:** Calculates total annual and monthly tax based on salary, bonuses, and deductions.  
* **Marital Status Options:** Supports different tax slabs for 'Single' and 'Married' individuals.  
* **Detailed Breakdown:** Displays a clear, easy-to-understand table showing how the tax is calculated across different slabs.  
* **Exemption Analysis:** Accounts for major deductions, including:  
  * Provident Fund (PF/EPF)  
  * Citizen Investment Trust (CIT)  
  * Social Security Fund (SSF)  
  * Life Insurance Premium  
  * Medical Insurance Premium  
* **Tax Optimization Tips:** Provides actionable suggestions on how to maximize contributions to retirement funds and insurance to lower tax liability.  
* **Informational Section:** Includes a collapsible section with up-to-date details on tax slabs and exemption rules for the fiscal year.  
* **Fully Responsive:** Modern UI that works seamlessly on desktops, tablets, and mobile devices.

## **Tech Stack**

This project is built with simplicity in mind, using only front-end web technologies:

* **HTML:** For the structure of the application.  
* **CSS:** For modern styling and a responsive layout.  
* **Vanilla JavaScript:** For all the calculation logic and DOM manipulation, with no external frameworks.

## **How to Use**

1. Clone or download this repository.  
2. Open the index.html file in any modern web browser (like Chrome, Firefox, or Edge).  
3. Fill in the form with your income and deduction details.  
4. Click the "Calculate My Tax" button to see the results.

## **Disclaimer**

The tax calculations are based on the rules and regulations for the **Fiscal Year 2081/82 (2024/25)** as provided by the Government of Nepal. This tool is intended for informational and estimation purposes only. Please consult with a qualified tax professional for official financial advice.
